# Personalized Networking Assistant

## Overview

The Personalized Networking Assistant is an AI-powered application that helps users improve professional networking through intelligent conversation generation, event analysis, and fact-checking. The project is built using FastAPI for the backend and Streamlit for the frontend.

## Features

- Event Analysis
- Conversation Generation
- Fact Checking
- RESTful API
- Interactive Swagger Documentation
- FastAPI Backend
- Streamlit Frontend

## Installation

Clone the repository:

```bash
git clone <repository-url>
```

Navigate to the project:

```bash
cd Personalized-Networking-Assistant
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the FastAPI server:

```bash
uvicorn main:app --reload
```

Open the API documentation:

```
http://127.0.0.1:8000/docs
```
